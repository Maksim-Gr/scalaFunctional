## Installation

catsFunctional is available on scala 2.11, 2.12, 2.13 and scalajs.

Add the following to your build.sbt
```scala
libraryDependencies ++= Seq(
  "com.github/Maksim-Gr" %% "catsFunctional-core" % "0.0.1")
```
